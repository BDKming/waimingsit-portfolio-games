# Wai Ming Sit Portfolio

Static production package for deployment.

## Entry

- `index.html` is the production entry page.
- `spinx.html` is kept as a direct alias of the same page.

## Deploy

Upload everything inside this folder to a static host, such as Netlify, Vercel static output, GitHub Pages, Cloudflare Pages, or any ordinary web server.

The large showcase videos are loaded from the configured remote R2 URLs, so this package only includes the local images required by the page.
